- **Design**: Modern dark-theme aesthetic

## Getting Started

### Prerequisites

- NodeJS (22+ recommended) installed on your machine.

### Installation

1. Clone the repository (or copy the files).
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the server:
```bash
npm start
```
The application will be available at `http://localhost:3000`.
